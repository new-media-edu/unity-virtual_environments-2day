# Session 01: Unity Workshop - Day 1

Date: Wednesday, April 22
Time: 10:00 am - 1:00 pm CET

## Session Focus

Session 01 introduces a no-code Unity workflow for artists who want to build virtual gallery mockups and native virtual environments. The focus is on spatial composition, basic modeling in ProBuilder, scene setup, and introductory spatial audio.

## Agenda

- Setup & Verification
	- Ensure Unity 6 or the current version is installed.
	- Confirm ProBuilder is installed and available.
- Gallery Mocking with ProBuilder
	- Create to-scale 3D white cube environments.
	- Practice extruding, face editing, and vertex manipulation.
	- Build core structures such as walls and pedestals.
- Building the Virtual Environment
	- Navigate the Scene view, Hierarchy, and Inspector.
	- Place assets with drag-and-drop workflows.
	- Import and position 3D models, textures, and images.
- Spatial Audio Basics
	- Add and position an Audio Listener on player or camera.
	- Configure Audio Sources with 3D Spatial Blend set to 1.0.
	- Tune Min/Max Distance so audio fades naturally with distance.
- Physical-to-Digital Demo
	- Overview of Polycam for mobile photogrammetry.
	- Compare Circular Scan for objects and Room Scan for spaces.

## Overnight Assignment

The Physical-to-Digital Bridge

- Software: Download Polycam, free version is sufficient.
- Task: Choose one physical object or a corner of a room at the Akademie.
- Capture: Perform a scan and generate a 3D mesh.
- Export: Upload and export as .obj or .glb for Session 02.

## Session Checklist

- Unity installed and opening correctly.
- ProBuilder installed and visible in Unity.
- One scanned asset exported as .obj or .glb.
- Any JPG/PNG images or MP4 videos prepared for use in the next session.
